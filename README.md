# SweetCup Template

A modern, responsive, and elegant **Vue 3 + Vite + Tailwind CSS** template designed for fast and beautiful web applications.

## 🚀 Live Demo

**View Live Demo**

![Sweet Cup Cake Preview](https://sweet-raiden-cupcake.netlify.app/)

## ✨ Features

- ⚡ **Vue 3** with Composition API + `<script setup>`
- ⚡ **Vite** – Lightning-fast build tool
- 🎨 **Tailwind CSS** for highly customizable styling
- 📱 Fully responsive design (Mobile-First)
- 🔄 State management with **Pinia**
- 🛠 Clean and maintainable project structure
- 🌙 Dark mode ready
- ⚙️ Modern development experience with hot reload

## 🛠 Tech Stack

- **Frontend**: Vue 3, Vite, Tailwind CSS
- **State Management**: Pinia
- **Styling**: Tailwind CSS + PostCSS
- **Build Tool**: Vite

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/FarhadJs/sweetcup-template.git

# Navigate to the project
cd sweetcup-template

# Install dependencies
npm install

# Start development server
npm run dev
